## Snowflake AI workspace

This repo currently contains two tracks:

- **Track A (implemented)**: Snowflake-backed **Streamlit dashboards** for ETF + securities exploration.
  - Start here: `docs/track-a/README.md`
- **ETF Intelligence Portal (spec only)**: A future Next.js + FastAPI portal described in `SPECIFICATION.md` (code folders like `web/` and `api/` are not in this repo snapshot yet).

Key files:

- `docs/track-a/README.md`: how to set up Snowflake objects and run the Streamlit dashboards
- `ETF_DB/`: Snowflake SQL scripts + Streamlit apps
- `SPECIFICATION.md`: implementation spec for the portal
